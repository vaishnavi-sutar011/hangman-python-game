# 🎮 Hangman Game (Python)

## 📌 Description
This is a simple Hangman game built using Python where users guess letters to find the hidden word.

## 🚀 Features
- Random word selection
- Hint system
- Limited attempts
- Win/Loss result

## 🛠️ Technologies
- Python

## ▶️ How to Run
Run the file:
python hangman_game.py
